## Commands

### `gtm webhook sync`

Regenerate webhooks/registry.yaml from live Modal + Hookdeck state.

```bash
uv run gtm webhook sync [OPTIONS]
```

### `gtm webhook list`

Print the cached registry as JSON. Errors if no cache; run `sync` first.

```bash
uv run gtm webhook list [OPTIONS]
```
